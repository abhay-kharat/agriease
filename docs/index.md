## Project Overview

Agriease is a full-stack agricultural platform designed to connect farmers, suppliers, and delivery agents. It features an e-commerce marketplace, equipment rental, AI-powered plant disease detection, and advanced delivery logistics.

## Project Structure

This project consists of 3 parts:

### backend (backend)

- **Type:** backend
- **Location:** `Agriease/backend`
- **Tech Stack:** Spring Boot, PostgreSQL, Razorpay
- **Entry Point:** `com.agriease.backend.BackendApplication`

### frontend (frontend)

- **Type:** web
- **Location:** `Agriease/frontend`
- **Tech Stack:** React, Vite, MUI
- **Entry Point:** `src/main.jsx`

### ai-backend (ai-backend)

- **Type:** backend
- **Location:** `Agriease/ai-backend/Flask Deployed App`
- **Tech Stack:** Flask, PyTorch, CNN
- **Entry Point:** `app.py`

## Cross-Part Integration

The Spring Boot backend acts as the central orchestrator, serving the React frontend and calling the Flask AI service for specialized image analysis.

## Getting Started

### backend Setup

**Prerequisites:** JDK 17, Maven

**Install & Run:**

```bash
cd Agriease/backend
./mvnw install
./mvnw spring-boot:run
```

### frontend Setup

**Prerequisites:** Node.js v18+

**Install & Run:**

```bash
cd Agriease/frontend
npm install
npm run dev
```

### ai-backend Setup

**Prerequisites:** Python 3.8+, Virtualenv

**Install & Run:**

```bash
cd "Agriease/ai-backend/Flask Deployed App"
pip install -r requirements.txt
python app.py
```
